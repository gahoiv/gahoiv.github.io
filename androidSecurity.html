Overview

  • App security best practices

Enforce secure communication

  • Use implicit intents and non-exported content providers
  • When sharing data between two apps that you control or own, use signature-based permissions. These permissions don't require user confirmation and instead check that the apps accessing the data are signed using the same signing key.
  • 
      <manifest xmlns:android="http://schemas.android.com/apk/res/android"
      package="com.example.myapp">
    <permission android:name="my_custom_permission_name"
                android:protectionLevel="signature" />
